# Sharajman Drupal Website

This repository contains the source code for the **Sharajman Drupal Website**.
The project is built using the **Drupal CMS** and runs on a local development environment using **XAMPP**.

## 🚀 Project Overview

Sharajman is a Drupal-based website designed to manage and display web content using the powerful Drupal content management system. It provides a flexible structure for creating pages, managing users, and organizing content.

## 🛠 Technologies Used

* Drupal CMS
* PHP
* MySQL
* HTML / CSS / JavaScript
* XAMPP (Local Development Environment)

## 📂 Project Structure

```text
sharajman-drupal/
│
├── core/               # Drupal core files
├── modules/            # Custom or contributed modules
├── themes/             # Website themes
├── sites/              # Site configuration and files
├── vendor/             # Composer dependencies
├── index.php           # Entry point of Drupal
└── composer.json       # Project dependencies
```

## ⚙️ Installation (Local Setup)

1. Clone the repository:

```
git clone https://github.com/md-rihan01/drupal-clinic.git
```

2. Move the project folder into your XAMPP directory:

```
C:\xampp\htdocs\
```

3. Start the following services from the XAMPP Control Panel:

* Apache
* MySQL

4. Create a database in phpMyAdmin.

5. Open the project in your browser:

```
http://localhost/drupal-clinic
```

6. Follow the Drupal installation wizard and connect your database.

## 📌 Requirements

* PHP 8.x
* MySQL
* Composer
* Apache Server (XAMPP recommended)
